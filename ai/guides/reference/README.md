# Template / style reference

The project uses the **oatmeal-olive-instrument** template.

- **Path:** `reference/oatmeal-olive-instrument/`
- **Design:** Olive color palette, Instrument Serif (display), Inter (sans). Tailwind theme in `oatmeal-olive-instrument/tailwind.css`.
- Base all app UI, layout, and components on the code in that folder.
- See `aiDocs/coding-style.md` for general conventions and Tailwind/theme usage.
